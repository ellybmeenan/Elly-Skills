# LinkedIn Post Writer — Claude Skill v1.0

A Claude skill that writes high-performing LinkedIn posts based on how the algorithm actually works in 2026.

Drop this into your Claude setup and it will handle hook writing, post structure, formatting, and tone — automatically matching the right format to your content type (tips, stories, hot takes, announcements).

## What's inside

- Hook formulas that stop the scroll
- Post structures for short, medium, and story formats
- Formatting rules to protect your reach
- What to avoid (engagement bait, links in body, over-posting)
- A drafting process that saves your writing style for every future post

Built for anyone who wants LinkedIn content that sounds human, travels far, and doesn't take an hour to write.

## How to install

Download the `.skill` file from the Releases section and add it to your Claude skills folder.
